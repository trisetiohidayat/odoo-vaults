# POS Mercado Pago

## Overview
- **Name:** POS Mercado Pago
- **Category:** Sales/Point of Sale
- **Depends:** `point_of_sale`
- **Author:** Odoo S.A.
- **License:** LGPL-3

## Description
Integrates your POS with the Mercado Pago Smart Point terminal. Enables customers to pay for orders via Mercado Pago card/UPI QR at the POS.

## Features
- Quick payment via swiping, scanning, or tapping card/QR code
- Supports credit/debit cards via Mercado Pago Smart Point terminal

## Data Files
- `views/pos_payment_method_views.xml` — Payment method configuration

## Assets
- POS frontend assets for Mercado Pago terminal integration
