# l10n_co - Colombia Accounting

## Overview
- **Name:** Colombia - Accounting
- **Country:** Colombia (CO)
- **Category:** Accounting/Localizations/Account Charts
- **Version:** 1.0

- **License:** LGPL-3

## Description
Colombian accounting and tax preconfiguration. Provides the Colombian chart of accounts aligned with NIIF (Normas de Informacion Financiera) and DIAN (Direccion de Impuestos y Aduanas Nacionales) requirements.

## Models
No Python models. Fully data-driven.

## Data Files
- `data/account_chart_template_data.xml` — Colombian chart of accounts template
- `data/l10n_latam.identification.type.csv` — Colombian identification types (NIT, Cedula, etc.)
- `demo/demo_company.xml` — Demo company
