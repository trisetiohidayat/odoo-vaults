# What's New in Odoo 17

Odoo 17 introduced major improvements across the framework and applications.

## Framework Highlights

### Discuss (formerly Mail)
- Channels renamed from `mail.channel` to `discuss.channel`
- Guest sessions: anonymous public portal users can now be part of live channels
- Improved notification system

### Studio & Customization
- Studio improvements for custom model creation
- Better view inheritance

### Web Client
- New Kanban improvements
- Form view enhancements
- Better mobile responsiveness

### ORM
- Improved performance on `search_read`
- Better caching with `@api.depends_context`

### Database
- PostgreSQL 16 support

## Key Module Changes

| Module | Odoo 16 → 17 Changes |
|--------|----------------------|
| sale | Improved quotation workflow |
| purchase | Vendor lead times |
| stock | Multi-warehouse improvements |
| account | E-invoicing improvements |
| mrp | Production scheduling |
