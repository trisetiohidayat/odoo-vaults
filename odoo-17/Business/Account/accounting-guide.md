# Accounting Guide

## Creating a Customer Invoice

1. Go to **Accounting -> Customers -> Invoices**
2. Click **Create**
3. Select customer, add invoice lines (products or manual entries)
4. Click **Confirm** to post the invoice
5. Use **Register Payment** to record payment

## Recording a Vendor Bill

1. Go to **Accounting -> Vendors -> Bills**
2. Click **Create**
3. Select vendor, add bill lines
4. Optionally match with a purchase order via **Add Credit Note** / PO linking
5. Click **Confirm** and **Post**

## Managing Payments

1. Go to **Accounting -> Payments** (or use the **Register Payment** button on an invoice)
2. Select journal (Bank, Cash, etc.)
3. Record payment date and amount
4. Reconcile with open invoices

## Journal Entries

1. Go to **Accounting -> Configuration -> Journals**
2. Select a journal (e.g., Miscellaneous Operations)
3. Click **Journal Entries** to create manual entries
4. Post the entry to update accounting records
