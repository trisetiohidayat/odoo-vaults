# New Modules in Odoo 17

## Enterprise Modules

New paid/enterprise modules in Odoo 17 include improvements to:
- `documents` — Document management
- `approvals` — Approval workflows
- `spreadsheet` — Spreadsheet engine
- `voip` — VoIP integration
- `event_social` — Social media event integration

## Community Modules

Key community additions:
- Improved `fetchmail` modules
- `google_calendar` improvements
- `microsoft_calendar` sync
