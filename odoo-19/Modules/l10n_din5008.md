## Technical Notes
- Country code: `de`, `ch` (Germany, Switzerland)
- Localization type: document layout (DIN 5008 standard)
- Auto-installs with `account` for DE/CH companies
- Custom model files: base_document_layout.py, res_company.py, res_config_settings.py

## Models

### `BaseDocumentLayout` (Extended, Transient)
Extends the document layout wizard with DIN 5008-specific fields.

**Fields:**
- `report_footer` — Custom footer (VAT excluded per DIN 5008 rules)
- `company_details` — Address formatted without empty lines
- `l10n_din5008_invoice_date` — Invoice date field (default: today)
- `l10n_din5008_due_date` — Due date (default: today + 7 days)
- `l10n_din5008_delivery_date` — Delivery date (default: today + 7 days)
- `street`, `street2`, `zip`, `city`, `company_registry`, `bank_ids`

**Key Methods:**
- `_default_report_footer()` — Returns company contact info without VAT (VAT is shown separately in DIN 5008 layout)
- `_default_company_details()` — Strips empty lines from address for strict DIN 5008 formatting
