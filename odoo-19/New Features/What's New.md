# What's New in Odoo 19

## Overview

Major new features and improvements in Odoo 19.

## 🚀 New Features

### 1. Studio Improvements
- Better drag-and-drop
- More customization options
- Improved view editor

### 2. Accounting
- Enhanced bank reconciliation
- PEPPOL invoicing
- Better tax handling

### 3. Manufacturing (MRP)
- Improved planning
- New subcontracting features
- Better workorder management

### 4. POS
- Faster performance
- Improved offline mode
- New payment integrations

### 5. Website
- New theme builder
- Better SEO tools
- Improved page builder

### 6. HR
- New attendance system
- Improved timesheets
- Better expense management

## 🔄 Improvements

| Area | Improvement |
|------|------------|
| ORM | Faster queries |
| Performance | Reduced load times |
| UI | Modern interface |
| Mobile | Better responsiveness |

## 📦 New Modules (vs Odoo 17)

- `auth_passkey` - Passkey authentication
- `cloud_storage` - Cloud storage integration
- `api_doc` - API documentation
- `html_editor` - Rich text editor
- `iot_base`, `iot_drivers` - IoT integration
- `partnership` - Partnership management
- `pos_glory_cash`, `pos_imin` - New POS hardware
- `project_stock` - Project inventory
- `sale_gelato` - New sale features
- `website_event_track_live` - Live event streaming

## ⚠️ Deprecated Modules

- Some legacy modules consolidated
- Old payment providers removed
