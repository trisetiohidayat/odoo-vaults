# Project Stock Landed Costs

## Overview
- **Name:** Project Stock Landed Costs
- **Category:** Supply Chain/Inventory
- **Depends:** `project_stock_account`, `stock_landed_costs`
- **Auto-install:** Yes
- **License:** LGPL-3

## Description
Technical bridge module that brings landed cost adjustments from stock pickings into the project profitability panel. Combines `project_stock_account` (picking costs in project profitability) with `stock_landed_costs` (valuation adjustments) so that landed cost changes are reflected in project cost tracking.
